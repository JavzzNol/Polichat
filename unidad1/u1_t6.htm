Introducir fórmulas

La parte importante de una hoja de cálculo, son las operaciones matemáticas, para ello se tiene capturado todo el volumen de datos que sirve como elemento fundamental en la ejecución de las operaciones, el objetivo de aplicar las fórmulas es automatizar el proceso de cálculo que produzca un resultado en forma de datos para ser analizados y graficados.

Reglas para escribir una fórmula correctamente

Una fórmula es una expresión matemática que conjunta varios elementos como son: variables, números, constantes, operadores, referencia de celdas relativas o absolutas y funciones predefinidas, todo ello relacionado de tal manera lógica que produzca un resultado.

La manera de escribir una fórmula dentro de la hoja de cálculo comienza por el signo =. Enseguida del signo igual vienen los números, operadores, constantes, referencias de celdas y funciones predefinidas que la conforman, para finalizar el proceso oprimir la tecla enter.

Una de las ventajas de utilizarlas en el procesamiento de datos es al momento de cambiar alguna información, la fórmula se ejecuta automáticamente mostrando al instante el nuevo resultado.

Existen algunas reglas fundamentales a considerar en el momento de escribir una fórmula. Entre las reglas más importantes se tienen las siguientes:

  1. El uso de paréntesis es importante, sí se desea realizar una operación en particular, basta con encerrar dicha operación entre ellos, se realizan las operaciones de los paréntesis más internos, hasta los externos.
  2. Tener en cuenta la prioridad de los diferentes operadores que conforman una fórmula, con base en una tabla de prioridades.
  3. El caso de operaciones con el mismo nivel de prioridad, se evalúan las operaciones de izquierda a derecha.

Operadores

Son símbolos en una expresión matemática que representa la operación que tiene que realizarse entre dos elementos, se agrupan en las siguientes categorías:

  • Aritméticos.- Principalmente realizan cálculos matemáticos con números.
  • Concatenación.- Se emplean normalmente entre cadena de caracteres o texto.
  • Comparación.- Utilizados para comparar valores numéricos o expresiones.
  • Lógicos.- Se aplican en la comparación de expresiones, en el resultado existen dos valores posibles: falso o verdadero.
