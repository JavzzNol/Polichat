Creación de una gráfica

Una gráfica es la representación por medio de una figura, es la relación de los datos de una hoja de cálculo, cuyo objetivo consiste en simplificar el análisis de la información, para tomar decisiones.

Crear un gráfico que represente de manera sencilla el contenido de la hoja de cálculo, requiere de una metodología, se describe el procedimiento para la creación de una gráfica (Carballero, 2013, pp. 30 - 33):

  • Definir cual es el objetivo o fin a lograr dentro de la organización.
  • Capturar los datos necesarios para lograr ese propósito.
  • Estructurar la información en diversas hojas de cálculo.
  • Elegir el tipo de gráfico adecuado al fin perseguido.
  • Crear una gráfica fácil de interpretar y con un diseño.
