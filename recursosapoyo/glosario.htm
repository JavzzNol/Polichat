Glosario

Aplicación

Programa informático diseñado para ser utilizado en un trabajo puntual y usuario final.

 

Argumento o parámetro

Valor que puede adoptar una función.

 

ASCII (American Standard Code for Information Interchange)

Código Americano para el Intercambio de Información, estandariza el uso de caracteres en el mundo.

 

Barra de fórmula

Elemento de la hoja electrónica de cálculo que muestra el contenido real de las celdas, también permite editar el contenido de las celdas.

 

Barra de herramientas o cinta de opciones

Agrupa en bloques de botones las acciones que son comunes entre ellas.

 

Caché

Memoria interna de alta velocidad, en ella se almacenan los programas de mayor uso.

 

Celda

En una hoja de cálculo, es la intercepción de una columna y una fila.

 

Celda Activa

Celda en la que se trabaja al momento, en la cual se está escribiendo.

 

CSS (Cascade Style Sheet)

Hoja de Estilo en Cascada, agrupan en un estilo gran variedad de formatos o clases, para facilitar el diseño de páginas web.

 

Código fuente

Grupo de etiquetas que conforman el diseño de una página web.

 

Editor

Programa que permite hacer modificaciones a un archivo determinado.

 

Entrada de datos

Espacio diseñado para el registro de información de tipo numérico, texto, imagen o archivo.

 

Fila

Conjunto de celdas ordenadas horizontalmente, se denotan por un número.

 

File Transfer

Transferencia de archivo, envío de archivos por medio de una aplicación de red.

 

Filtro

Búsqueda de datos que cumplan uno o varios criterios.

 

Formulario

Facilita el registro de información para ser enviada a una base de datos.

 

Frame

Marco, área de una ventana que muestra dentro de ella una página web.

 

Función

Fórmula predefinida en la hoja de cálculo electrónica que facilita un cálculo determinado.

 

Fórmula

Diseñada por el usuario y permite obtener un resultado que cumple con criterios determinados.

 

Gráfica

Imagen que representa datos numéricos para brindar una fácil interpretación.

 

GUI

Interfaz gráfico de usuario, sistema operativo que muestra en pantalla sus comandos por medio de gráficos, iconos y ventanas.

 

Hardware

Parte física, componentes, materiales físicos de todo equipo informático.

 

Hipervínculo

Unir mediante un vínculo a un texto o imagen con otra página web o archivo.

 

Hoja de trabajo

Es el área compuesta por las celdas de una hoja de cálculo electrónica.

 

Host

Servidor principal, computadora con una gran capacidad de almacenamiento y procesamiento de información, da servicio por medio de una red a otras computadoras.

 

HTML (HyperText Markup Language)

Lenguaje de Marcas de Hipertexto, lenguaje de diseño de páginas web que utiliza etiquetas.

 

HTTP (Hypertext transfer Protocol)

Protocolo de Transferencia de Hipertexto, protocolo de comunicación entre un servidor y un usuario en la red.

 

Interfaz

Interconexión de dos equipos para transmitir datos.

 

Iteración o bucle

Repetición continúa de una instrucción.

 

Libro de Excel

Es una hoja de cálculo electrónica, puede contener una o varias hojas de cálculo.

 

Link

Enlace, interacción que hay entre una página web y otra.

 

Multimedia

Diferentes medios de comunicación (imagen, sonido y video) para transmitir información.

 

Navegador

Programas que permiten la interacción entre el usuario y los sitios de Internet.

 

ObjectLink

Incrustar, identificador de objetos, por medio de su clase y nombre.

 

Operador

Símbolo aritmético o función, utilizado para el diseño de fórmulas.

 

Portal

Es el conjunto de páginas que muestran un sitio web.

 

Rango

Grupo de celdas específicas.

 

Red

Dos o más computadoras conectadas para compartir recursos materiales e información.

 

Relleno

Término utilizado para indicar la acción de llenar de color las celdas.

 

Script

Guion, documento escrito en código que ejecuta diversas instrucciones dentro de un programa de computadora.

 

Selección

Proceso mediante el cual se eligen varias celdas, para realizar una acción determinada.

 

Software

Parte blanda, programa informático que permite la realización de una o varias tareas.

 

Solver

Comando que busca el valor óptimo de una celda.

 

Tag

Etiqueta, que se aplica en código html, para un determinado diseño a la página web.

 

URL (Uniform Resource Locator)

Localizador de recursos uniforme, dirección electrónica de una página web.

 

Vista preliminar

Vista previa del documento antes de imprimir.

 

Web

Telaraña, es un término utilizado para indicar la conexión entre varias redes de computadoras en el mundo.
